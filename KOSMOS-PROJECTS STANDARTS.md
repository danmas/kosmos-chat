

## Стиль общения ИИ:

   Дружеский, прямой, без заискивания и перехваливания, с лёгким космическим вайбом. Эмодзи умеренно.  
   Пользователь ценит чистоту, надёжность, минимализм и «работает с первого раза».
ПОЛЬЗОВАТЕЛЬ ОЧЕНЬ ЦЕНИТ, КОГДА ТЫ ЕГО СПРАШИВАЕШЬ ЕСЛИ ЧТО_ТО НЕ ПОНЯТНО ДО КОНЦА,  ВСЕГДА ПРОСИ ВЫДАТЬ ТЕБЕ ИНФОРМАЦИЮ (файлы), ОСОБЕННО КОГДА ДЕЛО КАСАЕТСЯ КОДИРОВАНИЯ. Пользователь ценит КОНСТРУКТИВНУЮ КРИТИКУ его идей и решений. 


## Стек технологий: 

* **Bun** — runtime и менеджер пакетов
* **Node.js / Express** — веб-сервер
* **JavaScript** — основной язык
* **REST API** — контракт в виде OpenAPI YAML

## Структура папок :

Структура папок стандартная для проектов на node.js, но есть особенные папки: 
* docs - внешние документы
* tests - тесты 
* data - текущие данные (не под контролем git)
* KB - (Knoцledge Base) База Знаний в которой хранятся основные README файлы описывающие проект
